O que a aula destaca como uma característica fundamental da linguagem Java?

  • Programação funcional
  • Orientação a eventos
  • Programação estruturada
  • Orientação a objetos
  • Programação reativa

Qual é o papel da JVM (Java Virtual Machine) no ecossistema Java?

  • Compilar o código Java diretamente para a máquina
  • Interpretar o bytecode e executar o programa Java
  • Fornecer um ambiente de desenvolvimento integrado
  • Facilitar a escrita de código Java
  • Distribuir aplicativos Java para diferentes plataformas

Qual é a principal diferença entre o JDK e o JRE?

  • O JDK é usado apenas para executar programas Java, enquanto o JRE é usado para desenvolver programas Java
  • O JDK contém ferramentas para desenvolvimento e execução de programas Java, enquanto o JRE contém apenas as bibliotecas necessárias para execução
  • O JRE inclui ferramentas de desenvolvimento e o JDK não
  • O JDK é necessário apenas para aplicativos web, enquanto o JRE é necessário para aplicativos desktop
  • O JDK e o JRE são idênticos, sem diferenças significativas

Qual foi o uso inicial planejado para a linguagem Java?

  • Desenvolvimento de sistemas operacionais
  • Criação de jogos de alta performance
  • Desenvolvimento de dispositivos eletrônicos inteligentes
  • Aplicações de inteligência artificial
  • Bancos de dados relacionais
