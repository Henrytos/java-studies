O que foi destacado como essencial para evitar comprometer um projeto em Java?

  • Uso de bibliotecas externas
  • Escolha de um ambiente de desenvolvimento adequado
  • Entendimento da estruturação do código e escopos
  • Implementação de testes unitários
  • Conhecimento avançado de design patterns

O que é enfatizado como essencial ao definir variáveis em Java?

  • Iniciar com um número
  • Usar espaços para separar palavras
  • Iniciar com um caractere especial
  • Não utilizar palavras reservadas
  • Sempre usar letras maiúsculas

Qual padrão de nomenclatura é mencionado para variáveis e métodos em Java?

  • Camel case
  • Snake case
  • Pascal case
  • Kebab case
  • Upper case

Qual a principal diferença entre tipos primitivos e tipos wrappers em Java?

  • Tipos primitivos são classes, enquanto tipos wrappers são interfaces
  • Tipos primitivos são valores literais, enquanto tipos wrappers são classes que representam esses valores
  • Tipos primitivos são usados apenas para variáveis locais, enquanto tipos wrappers são usados para variáveis globais
  • Tipos primitivos são definidos pelo usuário, enquanto tipos wrappers são pré-definidos
  • Tipos primitivos são usados apenas em métodos, enquanto tipos wrappers são usados apenas em classes
