Qual foi um dos tópicos abordados na aula sobre operadores em Java?

  • Operadores lógicos e bit a bit
  • Operadores de incremento e decremento
  • Operadores de comparação
  • Operadores ternários
  • Operadores de atribuição e aritméticos

O que os operadores relacionais avaliam em programação?

  • A relação entre duas variáveis ou expressões, retornando valores inteiros
  • A relação entre duas variáveis ou expressões, retornando valores de ponto flutuante
  • A relação entre duas variáveis ou expressões, retornando valores booleanos
  • A relação entre duas variáveis ou expressões, retornando valores string
  • A relação entre duas variáveis ou expressões, retornando valores de caractere

Qual é uma função comum dos operadores unários em Java?

  • Incrementar, decrementar e inverter valores
  • Concatenar strings
  • Realizar operações matemáticas complexas
  • Realizar operações lógicas entre variáveis
  • Definir a visibilidade de métodos

O que os operadores lógicos permitem fazer em Java?

  • Realizar operações de conversão entre tipos de dados
  • Criar variáveis locais dentro de métodos
  • Alterar a precedência de operações matemáticas
  • Definir a visibilidade de métodos
  • Combinação de expressões booleanas para formar expressões lógicas maiores

Qual é a sintaxe correta para comentários de bloco em Java?

  • { }
  • / /
  • //
  • /* */
