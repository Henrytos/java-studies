  • Para facilitar a leitura de documentação
  • Para evitar erros de compilação por referências não resolvidas
  • Para ajustar a visibilidade de métodos

  • Para realizar operações de entrada e saída
  • Para garantir a comparação correta de objetos com base em seus valores
