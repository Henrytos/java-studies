Qual é a principal diferença entre estruturas condicionais simples e compostas em Java?

  • Apenas estruturas condicionais simples permitem o uso de operadores relacionais
  • Estruturas condicionais compostas podem ter múltiplas condições, enquanto simples têm apenas uma
  • Estruturas condicionais compostas são obrigatórias para validação de resultados
  • Estruturas condicionais simples são mais eficientes em termos de desempenho
  • Estruturas condicionais compostas não permitem a utilização de operadores lógicos

Qual é a principal finalidade do controle de fluxo ternário em Java?

  • Modificar a estrutura de um loop
  • Substituir a declaração de variáveis estáticas
  • Atribuir valores a variáveis de forma condicional
  • Definir a visibilidade de métodos
  • Implementar interfaces gráficas avançadas

Qual comando é importante para controlar a execução dos casos no switch case em Java?

  • continue
  • return
  • exit
  • break
  • stop

Analisando o código abaixo, temos uma estrutura for: ```java public class Main { public static void main(String[] args) { for (int i = 0; i < 5; i++) { System.out.println(i); } } } ``` É verdade que o laço acima irá imprimir os números de 0 a 5 na saída?

  • Verdadeiro
  • Falso

Qual é o propósito da estrutura try-catch em Java, conforme explicado pelo professor?

  • Gerar erros de propósito
  • Controlar fluxos esperados
  • Personalizar mensagens de erro
  • Manipular exceções inesperadas
  • Definir variáveis locais

Por que é importante entender a hierarquia das exceções em Java?

  • Para identificar e tratar corretamente diferentes tipos de exceções
  • Para evitar a declaração de variáveis globais
  • Para organizar melhor o código fonte
  • Para facilitar a leitura de documentação
  • Para executar operações matemáticas complexas
